Parameters

model string

The model to use for the batch job.

src BatchJobSource

The BatchJobSource of the batch job. Currently Vertex AI supports GCS URIs or BigQuery URI. Example: "gs://path/to/input/data" or "bq://projectId.bqDatasetId.bqTableId". Gemini Developer API supports List of inlined_request, or file name. Example: "files/file_name".
